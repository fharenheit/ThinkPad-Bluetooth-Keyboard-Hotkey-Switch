# 사용상 설명

```
0. Fn 키를 고정 시켜줍니다. 

 - 보통 Fn+Esc 로 쓰는데 이 모델은 불가능 했습니다.

 - Hooking 프로그램이라 다른 키보드 동시 사용시 영향 줍니다.(바로가기 아이콘으로 필요시한 로드 하세요)

1.다운로드:

 https://github.com/AlexeyBoiko/ThinkPad-Bluetooth-Keyboard-Hotkey-Switch/blob/master/ThinkPadBluetoothKeyboardHotkeySwitch.zip

위 링크 누르고 화면의 Download  클릭!

zip 파일 압축플고 시작 프로그램에 등록.

2. 자세한 내용

https://github.com/AlexeyBoiko/ThinkPad-Bluetooth-Keyboard-Hotkey-Switch

 F1 ~ F11 고정가능, F12는 안됩니다

아래아 한글 Ctrl,Alt, Function key 조합에서도 잘 동작하네요.

- 시작 프로그램 등록법

 1. 윈도키+R 

2. shell:startup

3. 시작프로그램 폴더에 프로그램 복사

-  바로가기 아이콘 등록

 필요할 때만 호출해서 쓰시고  고정 해제 하는 아래 명령어를 바로가기 아이콘으로 등록하시면

 껐다 켰다 하실 수 있습니다.

 >> taskkill /im ThinkPadBluetoothKeyboardHotkeySwitch.exe /f
```
